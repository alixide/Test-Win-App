# Test-Win-App

A simple calculator application for Windows developed using C#.

## Description

This is a Windows desktop calculator application built with C# and .NET Framework/WinForms. The calculator provides basic arithmetic operations in an intuitive and user-friendly interface.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clear and intuitive user interface
- Windows desktop application
- Built with C# and .NET

## Requirements

- Windows operating system
- .NET Framework (version to be specified)
- Visual Studio (for development)

## Development

This project is developed using:
- **Language**: C#
- **Framework**: .NET Framework
- **UI Framework**: Windows Forms (WinForms)
- **IDE**: Visual Studio

## Usage

Run the application to perform basic calculator operations through the Windows desktop interface.
